    Quantile-Based Stress Scenarios

    Mild Stress (target_or)
    UNRATE ≥ 75th percentile OR Retail ≤ 25th percentile

    Flags periods of either high unemployment or low retail traffic relative to historical patterns. Creates early warning indicators of potential market stress.

    Severe Stress (target_hard)
    UNRATE ≥ 75th percentile AND Retail ≤ 25th percentile

    Highlights combined macro and consumer-activity shocks as red-alert conditions requiring immediate attention. Identifies highest-risk periods for CRE investors.

    Why Quantile-Based Approach?

    Using quantile thresholds allows our model to adapt to changing economic conditions without requiring manual recalibration. These relative measures ensure stress detection remains consistent across different market cycles and geographic regions.

    Data Preparation Strategies

    Safe Split Function
    • Ensures both stress and non-stress classes appear in train/test sets
    • Prevents training on incomplete class distributions
    • Maintains proportional representation across splits
    from sklearn.model_selection import train_test_split

    def safe_split(X, y, test_size=0.2):
      return train_test_split(
        X, y, test_size=test_size, stratify=y
      )
    Oversampling Strategy
    • RandomOverSampler boosts representation of rare stress samples
    • Guarantees balanced learning without introducing class bias
    • Helps models learn from infrequent but critical stress conditions
    • Applied only to training data, preserving test set distributions
    from imblearn.over_sampling import RandomOverSampler

    def balance_classes(X_train, y_train):
      ros = RandomOverSampler(random_state=42)
      X_resampled, y_resampled = ros.fit_resample(
        X_train, y_train
      )
      return X_resampled, y_resampled
